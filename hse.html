🏛️ Об Университете

Национальный исследовательский университет «Высшая школа экономики» (НИУ ВШЭ) — один из ведущих университетов России, основанный в 1992 году. Сегодня ВШЭ — это университет мирового уровня, входящий в топ-100 лучших университетов мира в предметных рейтингах по компьютерным наукам.

Факультет компьютерных наук ВШЭ был основан в 2014 году при активном участии Яндекса. Это первый факультет в России, созданный в партнёрстве с крупной IT-компанией. Студенты получают не только фундаментальное образование, но и практические навыки, востребованные в индустрии.

💡 Факт: ВШЭ — единственный российский университет, входящий в топ-100 по компьютерным наукам в рейтингах THE и QS. Программы по Data Science и AI входят в топ-50 мира.

«Мы готовим специалистов, которые создают технологии будущего. Наши выпускники работают в Яндексе, Google, Microsoft и ведущих стартапах мира».

— Иван Аржанцев, декан ФКН ВШЭ

🏫 IT-факультеты ВШЭ

💻

Факультет компьютерных наук (ФКН)

Основан: 2014 год (совместно с Яндексом)

Направления: Программная инженерия, Прикладная математика и информатика, Data Science, ИИ

Направления: Экономика и статистика, анализ данных, финансовая аналитика

Факультет бизнеса и менеджмента

💻 Языки программирования в ВШЭ

Широкий спектр языков — от фундаментальных до самых современных

🐍

Python

Основной язык на ФКН. Используется на всех курсах: от основ программирования до машинного обучения и анализа данных. Главный язык для Data Science и AI.

Курсы: Программирование на Python, Анализ данных, Машинное обучение, Deep Learning
🦀

C / C++

Изучается для системного программирования, алгоритмов и структур данных. Используется в курсах по разработке игр, высокопроизводительных вычислений.

Курсы: Алгоритмы и структуры данных, Системное программирование, Разработка игр на C++
🌐

JavaScript / TypeScript

Основной язык для веб-разработки. Изучается в рамках фронтенд- и бэкенд-курсов. Активно используется в проектной деятельности.

Курсы: Web-разработка, Фронтенд-разработка, Node.js
☕

Java

Используется в курсах по корпоративной разработке, бэкенду и мобильной разработке (Android).

Курсы: Java-программирование, Корпоративные системы, Разработка под Android
#️⃣

C# / .NET

Изучается в рамках разработки десктопных и корпоративных приложений, а также разработки игр на Unity.

Курсы: Программирование на C#, .NET-разработка, Разработка игр на Unity
🔴

R / SQL

Активно используются в курсах по анализу данных, статистике и работе с базами данных.

Курсы: Статистика, Анализ данных, Базы данных
⚛️

Kotlin / Swift

Используются в курсах по мобильной разработке под Android и iOS соответственно.

Курсы: Мобильная разработка (Android/iOS)
🔷

Go (Golang)

Изучается в современных курсах по бэкенд-разработке, микросервисам и высоконагруженным системам.

Курсы: Бэкенд-разработка, Микросервисная архитектура

📋 Поступление в ВШЭ

Бакалавриат

  • ЕГЭ: профильная математика, русский язык, информатика/физика
  • Минимальные баллы: математика — 65, русский — 60, информатика — 70
  • Средний проходной балл: 280-300+
  • Дополнительные вступительные испытания (олимпиады, собеседования)

Магистратура

  • Портфолио достижений
  • Мотивационное письмо
  • Вступительные испытания (по направлениям)
  • Рекомендательные письма (не всегда обязательно)

Для иностранных студентов

  • TOEFL iBT: 90+ или IELTS: 6.5+
  • TORFL (русский как иностранный) — уровень B2
  • Стипендиальные программы для иностранцев
